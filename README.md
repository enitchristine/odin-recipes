# odin-recipes

This is a project to build a basic recipe website, with the guidance of The Odic Project. The objective of this project is to build a functional but minimal website, demonstrating the HTML knowledge that I have learnt so far.

This knowledge include:
- building a HTML boiler template
- writing the head, body, headings, paragraphs, lists and attributes with the appropriate tags
- linking pages and adding images
- organizing files
- committing with meaninggul and concise messages
